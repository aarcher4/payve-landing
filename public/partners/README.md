# Partner Logos

Place your partner logo images in this directory with the following names:

- `partner-1.png` - First partner logo
- `partner-2.png` - Second partner logo
- `partner-3.png` - Third partner logo
- `partner-4.png` - Fourth partner logo

## Image Requirements

- **Format**: PNG (recommended) or SVG
- **Aspect Ratio**: Any (images will scale to fit within the square containers)
- **Background**: Transparent backgrounds work best
- **Size**: Recommended minimum 200x200px for quality, but any size will work as images will scale automatically

The logos will automatically scale to fit within the square containers while maintaining their aspect ratio. The square outline and styling will be preserved.
